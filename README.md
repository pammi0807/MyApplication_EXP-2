# MyApplication_EXP-2

Aim

To develop an Android application that demonstrates drawing graphic primitives such as line, rectangle, circle, oval and text using Canvas.

Algorithm

1. Start Android Studio and create a new project.

2. Create a custom class extending View.

3. Initialize a Paint object.

4. Override the onDraw() method.

5. Use Canvas methods to draw shapes.

6. Draw a line using drawLine().

7. Draw a rectangle using drawRect().

8. Draw circles using drawCircle().

9. Draw an oval using drawOval().

10. Display text using drawText().

11. Set the custom view in MainActivity.

12. Run the application to display graphical shapes.
